🚀 Quick Start
📦 Installation
bash# Clone the repository
git clone https://github.com/anastanveer/portfolio.git

# Move into the project
cd portfolio

# Install dependencies
npm install
💻 Run Local Development Server
bashnpm run dev
Then open 👉 http://localhost:3000

⚠️ Important: Do NOT open .tsx, .js, or source files directly in the browser. This is a Next.js project — must be served via dev server or built export.


📦 Production Build
Generate static export:
bashnpm run build
Final deployable folder:
text📁 out/

☝️ Upload the contents inside out/ to Namecheap public_html — NOT the out folder itself.

✅ Correct Hosting Structure
textpublic_html/
├── 📄 index.html
├── ⚙️  .htaccess
├── 🤖 robots.txt

├── 📂 _next/

└── 📁 files/
